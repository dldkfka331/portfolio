3— when?

html과 css등을 배우면서 제일 재밌었던 때는 애니메이션을 만들었을 때 입니다. 왜냐하면 작은 효과로도 새로운 느낌을 줄 수 있었기 때문입니다. 시간이나 딜레이를 조절하고, 여러가지 timing-function으로 다른 효과를 주면서 마법처럼 변하는 애니메이션을 볼 수 있었습니다. 심심할때마다 코드펜을 보는 저의 애니메이션을 보여드리겠습니다.

fire animation

div 박스에 border-radius와 transform: rotate로 불꽃 모양을 만들고, 각각의 불꽃에 애니메이션과 딜레이 효과를 주어서 무한 반복 애니메이션을 만들었습니다. display: flex를 이용하여 불꽃이 body의 가운데에 위치하도록 변경하였습니다.

box animation

전체 박스 안에 6개의 div를 만들고 각각 transform: rotate, traslate를 주어서 정육면체의 모양을 만들었습니다. transform-style: preserve-3d를 주어서 3d 설정을 하였고, 전체 박스와 각 면에 따로 애니메이션을 주어서 계속 움직이며 늘어나고 줄어드는 박스를 만들었습니다.

heart animation

heart라는 div 박스 안에 3개의 div 박스를 만들어서 하트 모양을 만들었습니다. SCSS의 변수와 mixin을 사용해 보았고, TimelineMax를 이용해서 무한 반복 움직임 효과를 준 애니메이션입니다.

circle animation

circle이라는 div 박스에 가상 요소 2개를 만들어서 각각의 가상 요소에 box-shadow를 이용하여 원이 총 4개인 것처럼 만든 후, transform: rotate를 준 애니메이션입니다. cubic-bezier를 이용하여 색다른 움직임을 주었습니다.
